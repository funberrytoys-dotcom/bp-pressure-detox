## 1. Executive Summary

### 1.1 Продукт
PWA-приложение «2 в 1» для пациентов кардиолога-сомнолога Dr. Tatyana Gromyko:
- **💗 Дневник давления** — трекинг АД, пульса, AI-анализ, динамика
- **🌙 Цифровой детокс** — ритуал перед сном, белый шум, трекер сна

**Ключевой дифф:** AI-анализ данных → бот прогревает пациентов на консультацию / курс / закрытый канал. Не просто трекер — **воронка монетизации**.

|---------|----------|----------|
| 40+ женщины с гипертонией | Забывают мерить АД, не видят трендов | Автоматический анализ + триггер к врачу |

| Подписчики Татьяны в TG | Хотят больше персонализации | Прямая интеграция с доктором |

### 1.3 Бизнес-модель

- Базовый трекинг — бесплатно
- AI-анализ + рекомендации — бесплатно (для прогрева)

## 3. Функциональные модули

### 3.1 Модуль: Дневник давления (BP Diary)

**User Stories:**
- [ ] Как пациент, я хочу ввести АД за 10 секунд (телефон → числовая клавиатура)
- [ ] Как пациент, я хочу видеть цветовой статус записи (зелёный/жёлтый/красный)
- [ ] Как пациент, я хочу график за неделю/месяц/3 месяца
- [ ] Как пациент, я хочу PDF-отчёт для врача (экспорт)
- [ ] Как Татьяна, я хочу получать Telegram-уведомление о тревожных значениях

**Tech:**
- Ввод: `<input type="tel" inputmode="numeric">` → быстрый доступ с мобильной клавиатуры
- Графики: Canvas 2D (сейчас) → Chart.js или ApexCharts (v2)
- Правила риска: ESH/ESC гайдлайны (стадии АГ)
- Экспорт: jsPDF или html2canvas для PDF

### 3.2 Модуль: Цифровой детокс (Sleep Detox)

**User Stories:**
- [ ] Как пользователь, я хочу 1-кнопку «начать вечерний ритуал»
- [ ] Как пользователь, я хочу таймер с красивым визуалом (анимация)
- [ ] Как пользователь, я хочу выбрать звук для засыпания
- [ ] Как пользователь, я хочу будильник, который будит в лёгкой фазе
- [ ] Как пользователь, я хочу утренний журнал сна (оценка + заметка)

**Сценарий использования:**
1. 21:00 — push: «Через час сон. Начни детокс?»
2. 21:01 — включаю детокс → экран затемняется → чеклист
3. 21:01–22:00 — таймер обратного отсчёта, можно свернуть приложение
4. 22:00 — push: «Выключаю экран. Включи звук.»
5. 22:01 — выбираю «дождь» → звук через Web Audio API
6. 22:31 — таймер звука выключает (30 мин)
7. 07:00 — будильник (если включён)
8. 07:15 — push: «Как спалось? Оцени сон.»

**Звуковой движок:**
```
White  → random samples (-1..1)
Pink   → Voss-McCartney фильтрация
Rain   → white noise + steep LP @ 800Hz
Ocean  → brown noise + LFO на фильтре
Forest → bandpass + рандомные chirp-осцилляторы
Fan    → narrow BP @ 180Hz (циклический)
```

### 3.3 AI-анализ и рекомендации

**Алгоритмы:**
1. **Классификация АД** (ESH/ESC 2023):
   - Нормальное: <120/<80
   - Повышенное: 120–129/<80
   - Стадия 1: 130–139 или 80–89
   - Стадия 2: ≥140 или ≥90
   - Криз: ≥180 или ≥120

2. **Тренд-анализ:**
   - Скользящее среднее (7 дней)
   - Линейная регрессия (slope за 14 дней)
   - Колебания (SD за неделю)
   - Утренний всплеск (morning surge): сравнение 06-09 vs остальное время

3. **Sleep Score:**
   - Длительность (цель 7-8ч)
   - Регулярность (разница между днями ≤30 мин)
   - Эффективность (время в кровати vs фактический сон)
   - Субъективная оценка (звёзды)
   - Просыпания (количество)

4. **Корреляция BP ↔ Sleep** (v2):
   - Регрессия: как сон влияет на АД утром
   - Флаг: «Плохой ночной сон → высокое утреннее АД»

### 3.4 AI-Бот прогрева (Warmup Engine)

**Логика:**
```
IF bp_avg_sys >= 140 OR bp_avg_dia >= 90:
    stage = "high_risk"
    action = send_alert_to_doctor() + show_CTA_consultation()
    message = "⚠️ Ваше давление требует внимания специалиста"

ELIF bp_trend_slope > 3 AND records >= 5:
    stage = "rising_trend"
    action = show_CTA_course()
    message = "📈 Давление растёт. Рекомендуется курс коррекции"

ELIF sleep_avg < 6 OR sleep_stars < 3:
    stage = "sleep_concern"
    action = show_CTA_sleep_channel()
    message = "😴 Ваш сон нарушен. Есть решение!"

ELIF bp_ok AND sleep_ok:
    stage = "healthy"
    action = show_CTA_sharing()  # реферальная программа
    message = "✅ Отличные показатели! Расскажите друзьям"
```

**Прогрев-воронка:**
| День | Действие |
|------|----------|
| 1-3  | Записываю данные, получаю базовый анализ |
| 4-7  | AI показывает первые тенденции, мягкий CTA |
| 8-14 | При росте риска — push о консультации |
| 15-21 | Персональное сообщение от бота: «Ваши данные показывают...» |
| 22+  | Спецпредложение: «Консультация со скидкой 20%» |
